[vSky](https://vsky.social) is an alternative client for [Bluesky](https://bsky.app) powered by TypeScript, Vue 3, and Vite. It's currently in early development and is not ready for production use.

## Features
- [x] daisyUI
- [x] Auto dark mode
- [x] PWA
- [x] SSR
- [x] YouTube embeds
- [x] Spotify embeds
